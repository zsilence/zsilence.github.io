如何实现跳出括号

学习编程第二天，今天又遇到了一点小麻烦。
在自定义快捷键的时候出现了问题，按照李笑来老师的教程来做的，但是无法实现“enter”之后跳出括号的操作。

原因：他提供的代码缺少了一句！

  1. 首先编辑init script
    # move cursor across the ending symbols...
    EndingSymbolRegex = /\s*[)}>\]/'";:=-]/
    atom.commands.add 'atom-text-editor', 'custom:jump-over-symbol': (event) ->
    editor = atom.workspace.getActiveTextEditor()
    cursorMoved = false
    for cursor in editor.getCursors()
    range = cursor.getCurrentWordBufferRange(wordRegex: EndingSymbolRegex)
    unless range.isEmpty()
    cursor.setBufferPosition(range.end)
    cursorMoved = true
    event.abortKeyBinding() unless cursorMoved

  2. 编辑keymap
    'atom-text-editor:not([mini])':
    'ctrl-k': 'custom:jump-over-symbol'

  3. Reload即完成！
