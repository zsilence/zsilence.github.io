如何把nvm的环境变量添加到.bash_profile中

上次成功安装了nvm，附带着就有了npm，但是每次打开terminal都要重新安装！！

主要原因：没有将nvm环境变量放到.bash_profile中
结果就是每次打开nvm都要重新设置一下环境变量

解决方法：

  1. 在用Finder中找~/.bash_profile（需要打开隐藏文件）

  2. 用atom或者 textedit打开，将环境变量代码添加到最后即可。
    export NVM_DIR="$HOME/.nvm"
    [ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh" # This loads nvm

  3. 重新打开terminal.

如何实现跳出括号

学习编程第二天，今天又遇到了一点小麻烦。
在自定义快捷键的时候出现了问题，按照李笑来老师的教程来做的，但是无法实现“enter”之后跳出括号的操作。

原因：他提供的代码缺少了一句！

  1. 首先编辑init script
    # move cursor across the ending symbols...
    EndingSymbolRegex = /\s*[)}>\]/'";:=-]/
    atom.commands.add 'atom-text-editor', 'custom:jump-over-symbol': (event) ->
    editor = atom.workspace.getActiveTextEditor()
    cursorMoved = false
    for cursor in editor.getCursors()
    range = cursor.getCurrentWordBufferRange(wordRegex: EndingSymbolRegex)
    unless range.isEmpty()
    cursor.setBufferPosition(range.end)
    cursorMoved = true
    event.abortKeyBinding() unless cursorMoved

  2. 编辑keymap
    'atom-text-editor:not([mini])':
    'ctrl-k': 'custom:jump-over-symbol'

  3. Reload即完成！

1 如何在Mac OS X中安装nvm

发表于 2016-06-22

#1 如何在Mac OS X中安装nvm

如何在Mac OS X中安装nvm（全称：），本来安装过程应该很简单，但是由于配置问题，产生了一个小问题，花了一些时间查找解决方法。

安装nvm，

brew install nvm

之后显示已经安装成功，但是安装nodejs还是不行，显示

-bash: nvm: command not found

经过一番搜索，找到了问题，第二天成功。

原因：很可能是因为设置了密码，

1.因此用

curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.31.1/install.sh | bash

这样安装，也会显示安装成功，但是并不能用。

2.改用sudo

sudo curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.31.1/install.sh | bash

需要输入密码，然后执行成功。

3.再添加nvm环境变量，依次执行

export NVM_DIR="$HOME/.nvm" [ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh" # This loads nvm

4.检查是否安装成功

command -v nvm

显示

nvm

这样，就可以进行后面的工作了。
